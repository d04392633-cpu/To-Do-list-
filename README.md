# To-Do List на Go

Простое консольное приложение для управления списком задач, написанное на языке Go.

---

## Возможности

- Добавление задач
- Удаление задач по номеру
- Просмотр всех задач
- Выход из программы

---

 Технологии

- Go (Golang)
- fmt пакет (ввод/вывод)
- os пакет (завершение программы)
- map для хранения задач

---

## Как работает проект

Программа использует `map[int]string` для хранения задач, где:

- ключ — это номер задачи
- значение — текст задачи

При каждом добавлении индекс увеличивается автоматически.
